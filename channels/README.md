# inboundRabbitMQ / outboundRabbitMQ

Start Parameters:

--rabbitHost=&lt;hostname or IP&gt;

* default: "amq"
 
--rabbitUser=&lt;user-id&gt;

*  default: "user"
 
--rabbitPwd=&lt;password&gt;

*  default: "password"
 
--rabbitURL=&lt;url&gt;

*  overrules rabbitHost, rabbitUser and rabbitPwd
*  by default not set
