# inboundRabbitMQ / outboundRabbitMQ

Start Parameters:

    --rabbitHost=<hostname or IP>

* default: "amq"
 
    --rabbitUser=<user-id>

*  default: "user"
 
    --rabbitPwd=<password>

*  default: "password"
 
    --rabbitURL=<url>

*  overrules rabbitHost, rabbitUser and rabbitPwd
*  by default not set
